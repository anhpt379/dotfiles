# Customizations

## Name window with `CMD+E`

Settings -> System -> Shortcuts -> Name Window... -> Add -> `CMD+E`

## Search tabs with `CMD+/`

Settings -> System -> Shortcuts -> Tab search -> Add -> `CMD+/`
