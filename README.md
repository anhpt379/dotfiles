# dotfiles

## Installation

Clone this repo and use [make](https://www.gnu.org/software/make/) or
[stow](https://www.gnu.org/software/stow/) to install:

```bash
# Install everything
$ make install

# or install just a few components
$ stow git
$ stow vim
$ stow kitty
```
